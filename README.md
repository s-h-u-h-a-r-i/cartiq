## ShopFlow (Items Foundation)

SolidJS app foundation with:

- Google sign-in
- Modular structure (`auth`, `items`, shared `ui`) and SCSS styling

## Setup

1. Install dependencies:

```bash
npm install
```

2. Run:

```bash
npm run dev
```

## Scripts

- `npm run dev`
- `npm run build`
- `npm run preview`
